# Track 1: Exploring NF Tumor Data with cBioPortal

**Organized by the [Children's Tumor Foundation](https://www.ctf.org) and [Sage Bionetworks](https://sagebionetworks.org)**

---

## Overview

In this track, you will use [cBioPortal](https://www.cbioportal.org) — a free, browser-based tool — to explore genomic and clinical data from NF tumors. cBioPortal requires no coding and no account, making it accessible to anyone with a web browser.

## What You Will Do

- Navigate cBioPortal to find NF-relevant studies and datasets
- Explore mutation data, copy number alterations, and clinical annotations
- Create visualizations to compare patient groups and identify patterns
- Export data and figures for your own use

## What You Will Need

- A laptop with a modern web browser (Chrome or Firefox recommended)
- No accounts, no software installation

## Learning Objectives

By the end of this session you will be able to:

1. Find and select relevant studies in cBioPortal
2. Identify frequently mutated genes in NF tumors
3. Interpret basic genomic visualizations (OncoPrint, survival plots, mutation diagrams)
4. Form a biological question and use cBioPortal to begin exploring it
