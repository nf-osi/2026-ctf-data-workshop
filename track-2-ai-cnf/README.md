## Overview

Cutaneous neurofibromas (cNF) are benign skin tumors that affect the majority of people living with NF1. In this track, you will work through a published NF1 research study — starting from the paper, finding the data, and exploring it using **Claude Code**: an AI assistant that runs in your terminal and answers scientific questions by writing and executing code on your behalf.

You do not need to know R, Python, or any other programming language. You ask questions; Claude Code does the analysis.

The study you will work with is:

> **"Immortalization and characterization of Schwann cell lines derived from NF1-associated cutaneous neurofibromas"**
> Wallace et al., *PLOS ONE*, 2026. [https://doi.org/10.1371/journal.pone.0340183](https://doi.org/10.1371/journal.pone.0340183)

Researchers isolated Schwann cells from surgically removed cNF tumors, immortalized them, and used RNA-seq to confirm the immortalized lines retained key biological properties of the original cells. The data is publicly available on the NF Data Portal.

## What You Will Do

- Read a real NF1 research paper and trace its data back to the NF Data Portal
- Use Claude Code to explore the RNA-seq data by asking scientific questions in plain English
- Generate PCA plots, volcano plots, and gene set enrichment results (via Enrichr) without writing a line of code
- Dig into the biology: Schwann cell identity, the NF1/RAS pathway, drug targets, and more
- Produce a summary figure and plain-language interpretation you could share with a colleague

## What You Will Need

- A laptop with a modern web browser
- A free [Synapse account](https://www.synapse.org/register) — **please register before the workshop** (see [SETUP.md](../SETUP.md))
- Access to your workshop EC2 environment — URL and password provided at the door (no local software installation needed)

## Learning Objectives

By the end of this session you will be able to:

1. Navigate from a published paper to the underlying datasets on the NF Data Portal
2. Use Claude Code to perform and interpret RNA-seq analysis through plain language prompts
3. Interpret PCA plots, volcano plots, and gene set enrichment results in a biological context
4. Critically evaluate what RNA-seq data can and cannot tell us about NF1 cell line models
